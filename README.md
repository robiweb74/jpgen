# Euro Jackpot Generator

A premium, single-file web application that generates random numbers for the Euro Jackpot lottery.

## Features

- **Premium Design**: Features a modern, dark-themed UI with glassmorphism effects and gold accents.
- **Instant Generation**: Generates 5 main numbers (1-50) and 2 Euro numbers (1-12) instantly.
- **Visual Animations**: Smooth animations for revealing numbers.
- **Responsive**: Works beautifully on desktop and mobile devices.
- **Single File**: All HTML, CSS, and JavaScript are contained in a single `index.html` file for easy deployment and portability.

## Usage

1. Clone the repository or download the `index.html` file.
2. Open `index.html` in any modern web browser.
3. Click the "GENERATE NUMBERS" button to get your lucky numbers.

## Technologies

- HTML5
- CSS3 (Variables, Flexbox, Animations)
- Vanilla JavaScript
